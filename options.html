User Agent

Override the browser's User-Agent sent to websites. Useful for monitoring pages that serve different content to different browsers or block automation.

Quiet Hours

Suppress all notifications during the specified time window, regardless of channel or rule.

Times are in your local timezone. Set end time before start time to span midnight (e.g. 22:00 to 07:00).

Monitoring Rules

Each rule watches for a CSS selector on every page and fires a notification via the active channel when it appears.

Export / Import

Back up your configuration or transfer it to another machine. Credentials, rules, quiet hours, and channel settings are included. Auto-refresh state is not exported.

Each rule will contain "url": "" — fill in the target URL before running the daemon. Rules sharing the same URL are fetched once per cycle and evaluated together; the shortest checkIntervalSecs among them sets the reload rate.

Import

Importing will overwrite all current settings and rules.
